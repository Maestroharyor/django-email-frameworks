Email Template Application API

MJML Template Endpoint

URL: /apply/mjml/
Method: POST
Description: Apply using MJML email template

React Email Template Endpoint

URL: /apply/react-email/
Method: POST
Description: Apply using React Email template

Maizzle Template Endpoint

URL: /apply/maizzle/
Method: POST
Description: Apply using Maizzle template

Inky Template Endpoint

URL: /apply/inky/
Method: POST
Description: Apply using Inky template

Request Format

All endpoints expect a JSON payload with the following fields:

{
    "first_name": "string",
    "last_name": "string", 
    "job_posting_title": "string",
    "salary": "string"
}
